1. 클래스 기초 개념 정리

클래스는 객체를 생성하기 위한 설계도이다.
객체는 설계도인 클래스를 기반으로 메모리에 할당되는 인스턴스이다.

클래스의 구성 요소로는 변수필드와 생성자, 메소드가 있다.

2. 객체 지향 설계 맛보기
작성한 클래스는 CafeDrink로 카페 음료 클래스입니다.
음료는 메뉴 이름, 가격, 사이즈, 아이스 여부, 샷 개수정보를 가집니다.
그래서 정의한 클래스 변수는 아래 5개입니다.
menuName : String
price : int
size : String
isIce : boolean
shot : int
메서드는 각 변수의 값을 확인, 수정할 수 있는 getter와 setter 메서드를 작성했습니다.
추가로 음료의 정보를 출력해주는 printMenu()메서드를 작성했습니다.
